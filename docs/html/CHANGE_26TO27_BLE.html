BLE OM6626 vs OM6627(OM6629)

Overview

The following table shows the new features of the BLE module.

New Feature
Support coded phy
Support PA(periodic advertising) and synchronization
Support PAwR(periodic advertising with responses) and synchronization
Support connection establish by PA(periodic advertising)
Support PAST(periodic advertising sync transfer)

API Comments

ob_gap_period_adv_start Start periodicadvertising

ob_gap_period_adv_sync Sync to periodic advertising
ob_gap_period_adv_sync_cancel Cancel Sync to periodic advertising
ob_gap_period_adv_sync_terminate Terminate Sync to periodic advertising
ob_gap_period_adv_list_set Set Sync accept list of PA

ob_gap_period_adv_response_data_set Set response data of PA

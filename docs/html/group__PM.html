◆ pm_sleep_state_t

enum pm_sleep_state_t

Sleep state.

Enumerator
PM_SLEEP_ENTRY 

Event with sleep entering. The output of IO can be controlled.

PM_SLEEP_LEAVE_TOP_HALF_HSI 

Event with sleep leaving, CHIP is working on 'High Speed Internal Clock (RC32M)'. The output of IO can not be controlled.

PM_SLEEP_LEAVE_TOP_HALF 

Event with sleep leaving, CHIP is working on 'High Speed External Clock (XTAL32M)'. The output of IO can not be controlled.

PM_SLEEP_LEAVE_BOTTOM_HALF 

Event with sleep leaving, CHIP is working on 'High Speed External Clock (XTAL32M)'. The output of IO can be controlled.

PM_SLEEP_STORE 

Alias for PM_SLEEP_ENTRY, just for driver REG store.

PM_SLEEP_RESTORE_HSI 

Alias for PM_SLEEP_LEAVE_TOP_HALF_HSI, just for driver REG restore.

PM_SLEEP_RESTORE_HSE 

Alias for PM_SLEEP_LEAVE_TOP_HALF, just for driver REG restore.

◆ pm_status_t

enum pm_status_t

PM status.

Enumerator
PM_STATUS_ACTIVE 

All modules are alive, and CPU run with full speed.

PM_STATUS_IDLE 

All modules are alive, but CPU clock is gating.

PM_STATUS_SLEEP 

Power down most of module(CPU, Peripheral, etc), but 32K is alive, only gpio and sleep-timer can wake up chip

PM_STATUS_DEEP_SLEEP 

Power down most of module(CPU, Peripheral, etc), 32K is not alive, only gpio can wake up chip

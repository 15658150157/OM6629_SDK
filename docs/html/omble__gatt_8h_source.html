284/**@brief Send notification or indication, can only be called when acting as a GATT server
285 * @details After sending notification, when the data is sent, you will receive the @ref OB_GATT_EVT_TX_COMPLETE event, but before receiving this event, you can call ob_gatts_send_hvx continuously to cache the notifications to be sent to increase the communication rate\n
286 * @details After sending the indication, the peer device will respond to the @ref OB_GATTS_EVT_INDICATE_CFM event

Detailed Description

Tiny printf, sprintf and (v)snprintf implementation.

Optimized for speed on embedded systems with a very limited resources

)

Tiny snprintf implementation.

Parameters
[in]bufferA pointer to the buffer where to store the formatted string.
[in]countThe maximum number of characters to store in the buffer, including a terminating null character
[in]formatA string that specifies the format of the output
Returns
: The number of characters that COULD have been written into the buffer, not counting the terminating null character. A value equal or larger than count indicates truncation. Only when the returned value is non-negative and less than count, the string has been completely written.

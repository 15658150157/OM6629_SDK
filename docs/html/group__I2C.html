◆ drv_i2c_isr_callback()

void drv_i2c_isr_callback ( OM_I2C_Type *  om_i2c,
drv_event_t  event,
uint8_t *  data,
uint32_t  num 
)

The interrupt callback for I2C driver. It is a weak function. User should define their own callback in user file, other than modify it in the I2C driver.

Parameters
om_i2cThe I2C device address
eventThe I2C event
  • DRV_EVENT_COMMON_WRITE_COMPLETED
  • DRV_EVENT_COMMON_READ_COMPLETED
  • DRV_EVENT_COMMON_ABORT
  • DRV_EVENT_COMMON_ERROR
  • DRV_EVENT_COMMON_DMA2PERIPH_COMPLETED
dataThe data pointer of data to be read or write
numThe data buffer valid data count

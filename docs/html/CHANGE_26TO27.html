SDK Changes

  • The directory structure of the project has been optimized. Please click here for more details.
  • The default project is based on FreeRTOS. OS-less project is also available in "SDK/projects/ble_app_simple_nonrtos"
  • The default ROM BootLoader supports Secure Boot. Please click here for more details.
  • The location of the system library is changed and is split into separate library files. Please click here for more details.
  • Projects no longer depend on symbols in the ROM.
  • SRAM no longer reserves space for ROM at the end.
  • Keil/IAR/JLink plugin is updated, and new device name is OM6627X. Please click here for more details.
  • Many APIs have been added or changed.
  • The Flash partition is changed. Please click here for more details.
  • The Efuse partition is changed. Please click here for more details.

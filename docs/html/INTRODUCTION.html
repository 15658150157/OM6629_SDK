Introduction

This chip is a power-optimized true system-on-chip (SOC) solution for both Bluetooth low energy and proprietary 2.4GHz applications. It integrates a high performance and low power RF transceiver with Bluetooth base band and rich peripheral IO extension. This chip also integrates power management unit (PMU) to provide high-efficient power management. It targets 2.4GHz Bluetooth low energy systems, proprietary 2.4GHz systems, Human-Interface Devices (keyboard, mouse, and remote control), sports and leisure equipment, mobile phone accessories and consumer electronics.

The on-chip Bluetooth system compliant with version 5.4

The chip integrates up to 96MHz high-performance MCU, DMA, GPIO, SPI, UART, Timer, Watchdog, supports 32MHz external crystal, integrates multi-purpose max 12-bit ADC.

This chip integrates on chip 128kB or 256kB SRAM, 512bits EFUSE and supports user defined IDE system on chip SFLASH MCU development and JTAG software upgrade.

Features

Protocol Support

  • Bluetooth Low Energy (Compatible with Bluetooth 5.4)
  • 1Mbps, 2Mbps
  • Long range 500kbps, 125kbps
  • Bluetooth Mesh
  • Propriety 2.4-GHz

Radio Performance

  • -98dBm sensitivity @ 1Mbps GFSK
  • -95dBm sensitivity @ 2Mbps GFSK
  • TX power: -30 ~ +8dBm

MCU & Memory

  • ARM® Cortex™-M4 with floating-point unit, max 96MHz
  • 128kB or 256kB SRAM
  • 512-bit EFUSE
  • 1MB SFLASH

Clock

  • 32MHz & 32.768kHz crystals
  • 32MHz & 32.768kHz RC oscillators

PMU

  • 1.71V~3.6V single power supply
  • 3.8mA RX current (1Mbps GFSK)
  • 4.6mA TX current @ 0dBm output power
  • 1.2uA sleep current (16K RAM retention, RTC on)
  • Integrated DCDC BUCK Converter

Peripherals

  • Up to 38 General Purpose I/O pins
  • 8 channel DMA controller
  • 1 x UART with FIFO
  • 1 x UART with FIFO & Flow Control
  • 1 x UART with FIFO & LIN
  • 2 x I2C interface
  • 1 x I2S
  • 2 x SPI interface
  • 2 x QSPI interface
  • 1 x USB2.0 full speed
  • 1 x QDEC
  • 1 x LED PWM
  • 1 x 32-bit Timer/Counter with 4 Compare/ Capture/PWM channels
  • 2 x 16-bit Timer/Counter with 4 Compare/PWM channels
  • 1 x 32-bit Real Time Counter with 3 Compare channels
  • 1 x 16-bit Low Energy Timer with 4 Compare/ PWM channels
  • 1 x Watchdog Timer
  • 8 channel single-end 12-bit 1Msps ADC
  • Die temperature sensor with +/- 1.5 degree C accuracy after single-point calibration
  • 1 x Audio ADC
    • 16-bit ADC
    • SNR 93dB
    • 1 x DMIC (Digital Mic)
    • 1 x AMIC (Analog Mic)
    • Sample rate: 32KHz, 16KHz, 8KHz
  • Embedded IR Transmitter

Security and encryption

  • AES128/256
  • SHA256
  • True Random Number Generator
  • Secure boot in ROM

Package and Work Environment

  • -40 ℃~+85 ℃

Applications

This chip integrated circuit has a fully integrated radio transceiver and base band processor for Bluetooth® Smart. It can be used as an application processor as well as a data pump in fully hosted systems.

  • RF remote control
  • Electronic shelf labels
  • Smart home devices
  • Computer peripheral and I/O devices

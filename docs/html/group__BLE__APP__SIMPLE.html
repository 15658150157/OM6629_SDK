Detailed Description

Example for using bluetooth including servial services.

The following functions need to be called during initialization

  • drv_wdt_init(): Disable watch dog
  • board_init(): Initialize board
  • hardware_init(): Initialize the hardware
  • drv_rf_init(): Initialize RF driver
  • evt_init(): Initialize event system
  • omble_init(): Initialize the protocol stack configuration

then ble module initialize followed

  • app_adv_init(): Initialize the advertise module
  • app_sec_init(): Initialize the SMP (security manage protocol) module

then service initialize followed

  • service_common_init(): Initialize the GAP, GATT, BAT service
  • app_onmicro_dfu_init(): Initialize the OTA service
  • ancs_client_init(): Initialize the ANCS client
  • app_media_hid_init(): Initialize the media HID service
  • app_tspp_init(): Initialize the TSPP service

Example for multiple device aplication, including advertise, scan & connect.

The Multilink function can use the shell to interact with the device, and operate the device to advertise, scan and connect. Supported commands can be obtained by typing help through the shell

All supported commands below:

  • adv: Enable advertising as a peripheral device
  • scan: start scanning
  • conn: Connect other devices as a central device
  • disc: Disconnect

The specific parameters of each command are as follows

adv

Usage: adv start/stop [adv_index=0]

  • adv start: Used to turn on the advertise
  • adv stop: Used to turn off the advertise
  • adv_index: Used to specify the advertise index number (used in multi-broadcasting), the default is 0

Example:

adv start : Turn on advertise index=0
adv stop 1 : Turn off advertise index=1

scan

Usage: scan [timeout=3s]

After the scan is complete (timeout), the terminal will print out the scanned devices, and the output format is

index addr_type addr RSSI [ advertise name ]

Example:

scan 5

Output:

Starting scan, timeout 5 s
Scan Finished:
1. 1 C1:BF:66:26:A1:01 0 OM6626_BLE
2. 1 C1:BF:66:81:A1:01 0 OM6681_BLE

conn

Usage: conn index [timeout=5s]

index is the index value of the scanned device

Note: Scanning is required before connecting

Example: Connect to the device with index=0 in the scan result, and the timeout period is 5s

conn 0 5

Output:

Starting connect to (1) C1:BF:66:26:A1:01 OM6626_BLE, timeout 5 s
> OB_GAP_EVT_CONNECTED(0): 255
OB_GAP_EVT_CONNECTED
@ OB_GAP_EVT_CONNECTED
连接成功事件，参考 ob_gap_evt_connected_t
Definition: omble_gap.h:29

Where "OB_GAP_EVT_CONNECTED(0): 255" indicates that the link index is 0 and the broadcast index is 255 (unavailable)

If the connection fails, output:

Starting connect to (1) C1:BF:66:26:A1:01 OM6626_BLE, timeout 5 s
> Connection timeout

disc

Usage: disc index

index is the link index value

Example: disconnect from device with link index 0

disc 0

Output:

Disconnecting to idx=0 (1) C1:BF:66:26:A1:01 OM6626_BLE
> OB_GAP_EVT_DISCONNECTED(0): 0x16
OB_GAP_EVT_DISCONNECTED
@ OB_GAP_EVT_DISCONNECTED
连接断开事件，参考 ob_gap_evt_disconnected_t
Definition: omble_gap.h:31

◆ spi_control_t

enum spi_control_t

SPI Control.

Enumerator
SPI_CONTROL_CSN 

Set SPI controller CSn low/high, argu indicate csn value

SPI_CONTROL_DLY_SAMPLE_FE_SET 

When argu is NULL, disable falling edge delay sample and vice versa

SPI_CONTROL_DLY_SAMPLE_CYCLE_NUM_SET 

Set delay sample cycle num, argu is in the range [0, 3]

SPI_CONTROL_TX_INT_MASK_SET 

Set SPI tx total interrupt mask，1 is enable interrupt，0 is mask interrupt

SPI_CONTROL_RX_INT_MASK_SET 

Set SPI rx total interrupt mask，1 is enable interrupt，0 is mask interrupt

◆ spi_mode_t

enum spi_mode_t

SPI Mode.

Enumerator
SPI_MODE_0 

MODE0: the idle time is low and the sampling starts from the first hop edge.

SPI_MODE_1 

MODE1: the idle time is low and the sampling starts from the second hop edge.

SPI_MODE_2 

MODE2: the idle time is high and the sampling starts from the first hop edge.

SPI_MODE_3 

MODE3: the idle time is high and the sampling starts from the second hop edge.

SPI_MODE_NUM

drv_event_t  event,
uint8_t *  data,
uint32_t  num 
)

The interrupt callback for SPI driver. It is a weak function. User should define their own callback in user file, other than modify it in the SPI driver.

Parameters
om_spiThe SPI device address
eventThe driver uart event
  • DRV_EVENT_COMMON_TRANSFER_COMPLETED
  • DRV_EVENT_COMMON_READ_COMPLETED
  • DRV_EVENT_COMMON_ABORT
  • DRV_EVENT_COMMON_ERROR
  • DRV_EVENT_COMMON_DMA2PERIPH_COMPLETED
dataThe data pointer of data to be read or write
numThe data buffer valid data count
